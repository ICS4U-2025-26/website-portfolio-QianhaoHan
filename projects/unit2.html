Unit 2: Introduction to JavaScript

This unit covered core JavaScript concepts including variables, type conversion, conditionals, arrays, loops, DOM manipulation, and event handling through a hackathon-style summative project.

View GitHub Repository

Summative: Tech2Gether Budget & Access Planner

Task Description

Create a functional, interactive website that addresses a critical social issue facing Canadians. The project required demonstrating all JavaScript concepts learned in the unit: variables and calculations, type conversion with parseFloat(), conditionals (if/else if/else), arrays, loops, and comprehensive DOM manipulation including getElementById(), addEventListener(), textContent, innerHTML, and direct property access.

My Approach

I chose to address financial barriers in student robotics, inspired by my experience co-founding Tech2Gether. The app is a single-page application with four sections: a hero introducing the issue, a Budget & Risk Calculator that classifies team financial health as Safe/Caution/At Risk, a Season Planner with event and task management (including filtering, sorting, and progress tracking), and a Resources section that dynamically filters grants and links based on the team's risk level. All data is stored in JavaScript arrays and rendered to the DOM using loops that build HTML strings.

Budget Calculator Interface

The calculator features a two-column layout with income sources on the left and expenses on the right. All input fields are dynamically generated from JavaScript arrays using loops and innerHTML.

Code Snippet - Arrays & Dynamic Input Generation

// Arrays of income and expense categories (objects)
const incomeCategories = [
  { name: "School Funding", id: "income-school",
    placeholder: "e.g., 2000" },
  { name: "Sponsorships", id: "income-sponsors",
    placeholder: "e.g., 1500" },
  { name: "Fundraising", id: "income-fundraising",
    placeholder: "e.g., 800" },
  { name: "Grants", id: "income-grants",
    placeholder: "e.g., 1000" },
  { name: "Registration Subsidies", id: "income-subsidies",
    placeholder: "e.g., 500" },
  { name: "Other Income", id: "income-other",
    placeholder: "e.g., 200" }
];

// Loop to generate input fields from the array
function generateIncomeInputs() {
  let htmlString = "";
  for (let i = 0; i < incomeCategories.length; i++) {
    const category = incomeCategories[i];
    htmlString += `
      <div class="input-group">
        <label for="${category.id}">
          ${category.name} ($)</label>
        <input type="number" id="${category.id}"
          placeholder="${category.placeholder}">
      </div>
    `;
  }
  document.getElementById("income-inputs")
    .innerHTML = htmlString;
}

Risk Assessment Results

The calculator uses conditionals to classify teams into three risk levels based on their coverage ratio (income/expenses). Teams with 120%+ coverage are "Safe" (green), 80-119% are "Caution" (yellow), and below 80% are "At Risk" (red). Each level displays different recommended actions and resources.

Code Snippet - Budget Calculation & Risk Assessment

// Type conversion: parseFloat converts input strings to numbers
function calculateTotalIncome() {
  let totalIncome = 0;
  for (let i = 0; i < incomeCategories.length; i++) {
    const inputValue =
      document.getElementById(incomeCategories[i].id).value;
    const numericValue = parseFloat(inputValue);
    if (!isNaN(numericValue)) {
      totalIncome += numericValue;
    }
  }
  return totalIncome;
}

// Conditionals: classify risk level based on coverage ratio
function determineRiskLevel(coverageRatio) {
  if (coverageRatio >= 1.2) {
    return "Safe";
  } else if (coverageRatio >= 0.8) {
    return "Caution";
  } else {
    return "At Risk";
  }
}

// Main calculation function using DOM manipulation
function calculateBudgetHealth() {
  const totalIncome = calculateTotalIncome();
  const totalExpenses = calculateTotalExpenses();
  const remainingBudget = totalIncome - totalExpenses;
  
  let coverageRatio = 0;
  if (totalExpenses > 0) {
    coverageRatio = totalIncome / totalExpenses;
  }
  
  const riskLevel = determineRiskLevel(coverageRatio);
  const teamName =
    document.getElementById("team-name").value || "Your Team";
  
  // DOM manipulation: show results, update text
  document.getElementById("results")
    .classList.remove("hidden");
  document.getElementById("result-team-name")
    .textContent = teamName;
  document.getElementById("total-income")
    .textContent = "$" + totalIncome.toFixed(2);
  
  // Direct property access: change link href by risk level
  const resourceLink =
    document.getElementById("main-resource-link");
  if (riskLevel === "At Risk") {
    resourceLink.href =
      "https://www.firstinspires.org/robotics/frc/grants";
    resourceLink.textContent = "Apply for FIRST Grants Now";
  }
}

What I Learned

This project was a comprehensive application of every JavaScript concept from the unit. Building the budget calculator taught me how to use arrays of objects to store structured data and loops to dynamically generate HTML from them. The risk assessment system demonstrated the power of conditionals for branching logic, and the season planner pushed me to implement filtering, sorting, and CRUD operations on arrays. I also learned the distinction between textContent (for plain text) and innerHTML (for structured HTML), and how direct property access (like changing .href) can create responsive, context-aware interfaces. Connecting the project to Tech2Gether, my own non-profit, made the work personally meaningful.
