About These Games

These games were created using vanilla JavaScript to demonstrate:

  • Event handling - Responding to user clicks and input
  • DOM manipulation - Updating the page dynamically
  • Timers - Using setTimeout and setInterval
  • Canvas API - Drawing graphics and animations
  • Game loops - Using requestAnimationFrame for smooth rendering
  • State management - Tracking scores and game progress
