Learning Resources

A curated collection of helpful resources I used throughout the ICS4U course, organized by unit.

Unit 1: Programming Fundamentals

  • A Practical Introduction to Python Programming - Brian Heinold

    Comprehensive Python textbook covering fundamentals, file I/O, and more

  • Python Documentation - Input and Output

    Official Python docs on file handling and I/O operations

  • Real Python - Python Exceptions Guide

    In-depth guide to exception handling with try/except

Unit 2: Introduction to JavaScript

  • W3Schools - JavaScript Function Definitions

    Learn how to define and call functions in JavaScript

  • MDN - JavaScript Array Reference

    Complete reference for array methods and properties

  • JavaScript.info - Document Object Model (DOM)

    Learn DOM manipulation for interactive web pages

Unit 3: Algorithms, Machine Learning, and AI

  • GeeksforGeeks - Searching Algorithms

    Overview of linear search, binary search, and other searching techniques

  • Google Machine Learning Crash Course

    Free introductory course to machine learning concepts and TensorFlow

  • Elements of AI - Free Online Course

    Beginner-friendly introduction to AI concepts from University of Helsinki

Unit 4: Object-Oriented Programming

  • Real Python - Object-Oriented Programming in Python 3

    Comprehensive guide to classes, objects, inheritance, and more

  • Oracle Java Tutorials - Object-Oriented Concepts

    Official Java documentation on OOP principles

  • GeeksforGeeks - Python OOPs Concepts

    Classes, inheritance, polymorphism, and encapsulation in Python
